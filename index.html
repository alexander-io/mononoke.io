Atom Snippets, November 6 | 2016

github.com/alexanderclaybeard/atom-snippets

"Snippets are an incredibly powerful way to generate commonly needed code syntax from a shortcut." Learn more about snippets at this link ∴

Inside of the Atom Text editor, the snippets.cson file can be accessed through the Menu > Atom > Snippets...



Snippets are written in Coffee Script Object Notation, thus each snippet must adhere to a specific syntax. This consisits of 4 elements :
  • scoope, the language in which the snippet will work
  • name
  • prefix, the "trigger"
  • body, fully generated body of text 

'.source.java':
	'system print':
		'prefix': 'sysprint'
		'body': 'System.out.print($1);'

Originally, the term "carriage return" referred to a mechanism or lever on a typewriter. It was used after typing a line of text and caused the assembly holding the paper (the carriage) to return to the right so that the machine was ready to type again on the left-hand side of the paper.
